# Simulador de Sistema de Cobranza - C#

Aplicación desarrollada en C# utilizando Windows Forms que simula la atención de una caja mediante la estructura de datos Queue (FIFO - First In, First Out).

## Funcionalidades

- Agregar clientes a la cola
- Validación de campos obligatorios
- Validación numérica del importe
- Cobro respetando orden FIFO
- Control de cola vacía
- Limpieza automática de campos

## Tecnologías utilizadas

- C#
- .NET Framework
- Windows Forms
- Estructura de datos Queue<T>

## Conceptos aplicados

- Programación orientada a objetos
- Manejo de eventos
- Validaciones
- Estructuras de datos
